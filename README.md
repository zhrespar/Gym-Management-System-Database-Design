## University & Private Gym Database Management System

Relational database system designed for managing university and private sports centers.
Developed using MySQL 8.x with full normalization, business rule enforcement, and advanced SQL operations.
## Overview

This project models a complete gym management ecosystem including:

Member registration

Membership packages

Payment tracking

Class scheduling

Staff & trainer management

Revenue reporting

The system contains 11 relational entities and 30+ records per table, built following normalization principles and ER modeling standards.
